# Automatize Contabilidade — Landing Page

Redesign do site institucional da **Automatize Contabilidade** — light mode, sóbrio e editorial (referência Qonto), construído com HTML, CSS e JavaScript puros.

## Identidade
- **Cores:** Roxo `#6C2EF3`, Azul-marinho `#190F59`, Fundo `#F6F6F6` (Style guide oficial)
- **Tipografia:** Poppins (pesos 200/300)
- **Logos:** variantes em `assets/`

## Estrutura
```
index.html                → página principal (hero, sobre, soluções, blog, contato…)
styles/main.css           → design system e responsividade (360px+)
scripts/main.js           → menu, dropdown, formulários, modal, cookies, animações
publicacoes/              → páginas de artigos do blog
assets/                   → logos e imagens
```

## Como rodar
Abra `index.html` no navegador. Não há build nem dependências.

## Seções
Navegação fixa · Hero · Segmentos atendidos · Sobre nós (estatísticas) · Materiais (e-book) · Soluções · Publicações · Contato · Rodapé · Banner de cookies.
